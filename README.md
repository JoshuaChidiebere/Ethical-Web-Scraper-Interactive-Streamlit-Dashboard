# Ethical Web Scraper + Streamlit Dashboard

**Portfolio Project 2** — Scrapes public data responsibly and displays it in an interactive dashboard.

## Features
- Polite web scraping with requests + BeautifulSoup
- Data cleaning and storage
- Interactive Streamlit dashboard with filters, Plotly charts, and export options
- Search & multi-select filters

## Technologies
Python • requests • BeautifulSoup4 • pandas • Plotly • Streamlit

## How to Run
1. `pip install -r requirements.txt`
2. `python scraper.py` (to collect data)
3. `streamlit run dashboard.py`
